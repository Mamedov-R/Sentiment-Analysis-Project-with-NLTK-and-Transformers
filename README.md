# Sentiment-Analysis-Project-with-NLTK-and-Transformers
Sentiment Analysis project using NLTK and Transformers to classify text data as positive, negative, or neutral. Preprocessing, tokenization, and model fine-tuning applied to extract insights from textual datasets efficiently.
### Dataset
The `Reviews.csv` dataset (286 MB) is **not included** in this repo due to size limits.  
You can download it from [Kaggle/Other Source](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews).
